A completely custom-built, full-stack accommodations platform designed entirely around finding proper PG (Paying Guest) rooms strictly localized across major tier-1 and tier-2 cities.

### Key Functionalities

*   **Search Engine Integration:** Engineered customized city-based heuristic search queries that correctly identify nearby PG accommodations relative to user coordinates instantly.
*   **Secure API:** Deployed an encrypted, monolithic Mongoose and Express.js backend specifically designed to lock out insecure connections via rigorous backend authentication frameworks.
*   **Scalability:** Implemented a non-relational MongoDB structure to natively handle rapidly expanding database ingestion without relying on strict schemas, drastically reducing latency speeds.

The outcome delivered a highly stable, responsive frontend paired with an extraordinarily fast backend retrieval node setup prioritizing real-world user speed and simplicity above all else.
